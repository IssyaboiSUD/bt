# 📄 How to Convert the Guide to PDF

## Option 1: Browser Print (Easiest)
1. Open `BAYREUTH_APARTMENTS_GUIDE.html` in any web browser
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
3. Select "Save as PDF" as the destination
4. Click "Save"

## Option 2: Online Converter
1. Go to a website like html-pdf-converter.com
2. Upload the `BAYREUTH_APARTMENTS_GUIDE.html` file
3. Download the generated PDF

## Option 3: Command Line (Advanced)
If you have Node.js installed:

```bash
# Install wkhtmltopdf (if not already installed)
# On Mac: brew install wkhtmltopdf
# On Ubuntu: sudo apt-get install wkhtmltopdf

# Convert HTML to PDF
wkhtmltopdf BAYREUTH_APARTMENTS_GUIDE.html BAYREUTH_APARTMENTS_GUIDE.pdf
```

## Option 4: VS Code Extension
1. Install "Print HTML" extension in VS Code
2. Open the HTML file
3. Right-click and select "Print HTML"
4. Choose "Save as PDF"
