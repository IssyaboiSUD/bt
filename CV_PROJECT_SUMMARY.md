# 🏠 Bayreuth Apartments - Vacation Rental Booking Platform

## Project Summary (CV Version)

**Full-Stack Web Application** | Next.js, React, TypeScript, Stripe, Prisma | 2024

**Developed a complete vacation rental booking platform** enabling guests to browse properties, book dates, and process secure payments while providing automated confirmation emails and booking management for hosts.

### 🎯 Key Achievements
- **End-to-End Development**: Built complete booking system from concept to deployment
- **Real Payment Processing**: Integrated Stripe Checkout for secure online transactions  
- **Automated Workflows**: Implemented email notifications and booking confirmations
- **Responsive Design**: Mobile-first approach with modern UI/UX
- **Database Management**: Designed and implemented data models for bookings, users, and payments

### 🛠️ Technical Stack
**Frontend**: Next.js 15, TypeScript, Tailwind CSS, React Hook Form, React DatePicker  
**Backend**: Prisma ORM, Next.js API Routes, Stripe Webhooks  
**Payment**: Stripe Checkout, Nodemailer, SMTP Integration  
**Deployment**: Vercel, Environment Variables, SSL/HTTPS

### 📋 Core Features
- **Interactive Photo Gallery**: Booking.com-style image viewer with modal
- **Real-time Price Calculation**: Dynamic pricing based on dates and guests
- **Secure Payment Processing**: Stripe integration with webhook handling
- **Automated Email System**: Booking confirmations and host notifications
- **Database Architecture**: User management, booking tracking, payment records
- **Responsive Design**: Mobile-optimized interface across all devices

### 🔧 Technical Implementation
- **Database Schema**: Users, Apartments, Bookings, Payments with proper relationships
- **API Architecture**: RESTful endpoints with comprehensive error handling
- **Security Features**: Environment variables, input validation, SSL/HTTPS
- **Performance**: Image optimization, code splitting, caching strategies

### 📊 Project Metrics
- **8 Pages**: Complete user journey implementation
- **2 Apartments**: Multi-property management system  
- **3 Database Models**: Comprehensive data architecture
- **5 API Endpoints**: Full backend functionality
- **100% Responsive**: Mobile-first design approach

### 🚀 Deployment & DevOps
- **Vercel Platform**: Automated deployment pipeline
- **Environment Management**: Development vs production configurations
- **Domain Configuration**: Custom domain with SSL certificate
- **Security**: Enterprise-level security implementation
